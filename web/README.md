# Habit Tracker Web

A clean Next.js app isolated under `web/`, separate from the mobile app.

## Run locally

```bash
cd web
pnpm install   # or: npm install
pnpm dev       # or: npm run dev
# open http://localhost:3000
```

If port 3000 is in use:

```bash
PORT=3001 pnpm dev
```

## Build/Start

```bash
pnpm build
pnpm start
```
